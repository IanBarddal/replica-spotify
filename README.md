SOBRE

Projeto fullstack desenvolvido com o objetivo de praticar o uso de Node.js, Express.js e React.js, sem fins lucrativos. Através de uma integração com a API do Spotify, é possível escutar seus artistas, músicas, playlists e álbuns favoritos, como se estivesse usando o software original. Além disso, também pode-se fazer buscas, parear aparelhos, receber recomendações e lidar com favoritos.
